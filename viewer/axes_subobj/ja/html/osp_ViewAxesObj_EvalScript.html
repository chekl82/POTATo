Viewer用 Object : Script


概要

 

Axes内 Object描画関数。
描画時にデータとして、Scriptを設定します。
ViewerはObjectの描画時にScript を実行します。

  

主な用途

 

  1. Axisのコントロール。
  2. 簡単なObjectの描画。

 

変数一覧

 

Scriptで使用可能な主な変数は、次の通りです。
変数名内容
curdataカレントデータ
obj{idx}View-Axis-Object (idxは現行のVAO==Scritを示す)
handles主なハンドル
h.axesgcaと同じ
(基本的にはView-Axis-Object == control で使用可能な変数と同じです。)

 

Script例 1

 

例としてデータの大きさを表示し、AxisをTightにするスクリプトを示します。 

          f=curdata.gcf;                                                   
          [hdata,data]=osp_LayoutViewerTool('getCurrentDataRaw',f,curdata);
          text(0,0.1,['Data Size is ' num2str(size(data))]);               
          axis('tight');
