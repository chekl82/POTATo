Platform for Optical topography Analysis Tools
用語集
  

 

本書、Platform for Optical topography Analysis Tools のマニュアルで利用する 用語・略語を示します。

 
 

用語集
  
A  B  D  E  H  I  M  N  P  R  S  ア  カ  サ  タ  ハ  マ  ヤ  ラ 
 
 
 

A
  
用語 説明
Analysisデータ Preで作成・編集したデータ。対象となる実験データを示す情報と、ひとつの解析データに対するレシピを持つデータ。
 
 
 

B
  
用語 説明
Batchサブ状態 Pre状態において、同じレシピを持つ複数のデータを扱っている状態。
フィルタに関するレシピを一括編集出来る。
Blocking 刺激区間のデータを取得し、形式を整えること。
 
 
 

D
  
用語 説明
Developersモード プラットフォーム開発者向けモード。通常利用しない。
 
 
 

E
  
用語 説明
Editサブ状態 Summary状態において、Summary Statisticデータを新規作成する際に利用するサブ状態。
ETGフォーマット ETGから出されるデータファイル。
 
 
 

H
  
用語 説明
HBデータ ヘモグロビンの濃度変化に比例する量。
 
 
 

I
  
用語 説明
Inion 後頭隆起点。
 
 
 

M
  
用語 説明
Mixedサブ状態 Pre状態において、異なるレシピを持つ複数のデータを扱っている状態。
フィルタに関するレシピを統合出来る。
 
 
 

N
  
用語 説明
Nasion 鼻根点。
Newサブ状態 Summary状態において、Summary Statisticデータを編集する際に利用するサブ状態。
Normalモード 一般的な解析手法を実験データに適用するためのモード。
 
 
 

P
  
用語 説明
P3 Platform for Optical Topography Analysis Tools Version 3の略。
Platform for Optical Topography Analysis Tools 光トポグラフィ解析ツール環境。
システム名称。
post-stim 刺激終了後の時間。
POTATo Platform for Optical Topography Analysis Toolsの略。
POTAToデータ システムがユーザに公開しているデータ。
実験データから要約統計量の計算までの間のヘモグロビン濃度変化量等を示す。
Pre Preprocess状態の略。
Pre Auricular 耳介前点。
Preprocess状態 統計解析に対する前処理。
各実験データにノイズ除去等の設定を行う。
結果はAnalysisデータになる。
pre-stim 刺激を与える前の時間。
比較のために利用する。
 
 
 

R
  
用語 説明
RAWデータ 吸光度。
Researchモード 解析手法を設定し、解析を行うモード。
 
 
 

S
  
用語 説明
Singleサブ状態 Pre状態において、ひとつのデータを扱っている状態。
フィルタに関するレシピを編集出来る。
Stat Statistical Test状態の略を記載している場合がある。
Statistical Test状態 Summary状態で作成したSummary Statisticデータを用いて、統計的検定を行う。
stimulation 刺激のこと。
Summary Summary Statistics Computation状態の略を記載している場合がある。
Summary Statistics Computation状態 要約統計量の計算を行う。
Preで作成したAnalysisデータから平均や統計などの要約統計量を計算し、Summary Statisticデータとして保存する。
Summary Statisticsデータ Summary状態で計算された、要約統計量を格納したデータ。
 
 
 

ア
  
用語 説明
イベント 一瞬で与えたとみなせる刺激を扱う際の刺激モード名。
オプトード 光学センサ。本書では光トポグラフィ装置における光源および検出器。
位置データ 計測位置に関する情報全般。実験データに付加される。
 
 
 

カ
  
用語 説明
解析準備 Preprocess状態で行う処理。
解析データ Analysisデータの和名。
グループ解析 被験者間解析。
コンテキストメニュー 本書では右クリックにて開くメニューを指す。
区間データ 計測データから刺激情報によりデータを区切り，分離・取得したデータ。
 
 
 

サ
  
用語 説明
実験データ 光トポグラフィの実験に関わるデータ。
刺激 実験において被験者に与えた刺激。
刺激の種類 実験時に実施した刺激の種類。
刺激モード イベント/ブロックのいづれか。
刺激期間 実験において、刺激を与えていた期間。
刺激後の時間 刺激終了後の時間。
刺激前の時間 刺激を与える前の時間。
比較のために利用する。
 
 
 

タ
  
用語 説明
統計学的検定 Statistical Test状態で行う処理。
 
 
 

ハ
  
用語 説明
プロジェクト データを保存するディレクトリ。
ブロッキング 刺激区間のデータを取得し、形式を整えること。
ブロック ある期間で与えたとみなせる刺激を扱う際の刺激モード名。
ポップアップメニュー 本章ではMATLAB uicontrolのpopupmenuスタイルのGUIコンポーネントを示す。
他にはコンボボックス、プルダウンメニュ等とも呼ばれるGUIコンポーネント。
ホルダー 光トポグラフィの計測用に用いるオプトードを固定するための入れ物。
光トポグラフィ解析ツール環境 Platform for Optical Topography Analysis Toolsの和名 。
 
 
 

マ
  
用語 説明
メインウィンドウ POTATo起動後に立ち上がるメインメインウィンドウを示す。
 
 
 

ヤ
  
用語 説明
要約統計量の計算 Summary Statistics Computation状態で行う処理。
要約統計量データ Summary Statisticデータの和名。
 
 
 

ラ
  
用語 説明
レイアウト 描画方法を示す。
レシピ POTAToにおける解析手順を示すデータ。
連続データ 計測データそのものを指し、通常複数の刺激ブロックからなる。
