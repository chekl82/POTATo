Filter - Plugin : メモリガーベッジ関数


概要

 

Filter−Plugin関数.
メモリガーベッジを行うPlugin関数で, データを変更することはありません.
メモリガーベッジとは,断片化されたメモリ領域を整理し まとまったメモリ領域を確保することです.
処理は通常の PACK 命令と同様です.
メモリガーベッジの詳細は PACK もしくはWebページをご覧下さい.

  

主な用途

 

Out of memory Errorが頻繁に発生する場合に ご利用ください.
本システムにおいて顕著に断片化される処理は Viewerおよび,(Single/Batch)Recipe編集処理です. その為,大きなデータやメモリの小さいPCで解析を行い, Out of memory Errorが頻繁に発生する場合に ご利用ください.

データの変更は行わないため, どのデータに対しても実行でき, 複数回実行しても結果に影響は与えません.
また,設定は不要です.

 

制約

 

本Pluginは MATLAB R2006aを対象に実装しております.
MATLABのアップグレードにより PACK関数は,コマンドライン上以外では実行不可能になりました. このため,MALTABの上位バージョンでは実行不可能で警告のみ頻発するようになります.
この場合はこの関数は使用できませんのでご注意ください.

 

メモリ量の確保について

 

メモリガーベッジは現状のメモリ内の空きスペースを整理し 再利用できる形に変更します. 使用されているメモリや,メモリの絶対量を増やすものではありません.
そのため,Out of memory Errorの根本的な解決にはならず, メモリ容量自体を増やすことが必要になります.
メモリサイズの変更方法はの詳細はMATLABのヘルプをご覧下さい.
メモリ容量を増やす方法には,

  1. メモリを増設する方法,
  2. メモリの代わりにHDDを代用する方法.
が考えられます.
後者の方法はOSの設定で行うことが出来ます. MATLAB Release 2006a, Windows XP の場合, マイコンピュータを右クリックし,仮想メモリを多めに設定する ことにより見かけのメモリ容量を増やすことが出来ます.
